Yahtzee

Our first group project was to build a web based Yahtzee game. We started of with a HTML table filled with inputs (yay, fun... especially before learing how to use Emmet...) and applied styling with CSS. After learning about how to manipulate the DOM with JS we set up event listeners and built a basic summing function. After our lecture in objected oriented programming we started to transfer our more straight-forward code to a class-based version. A real challenge if you've never dealt with OOP before.

I learned a lot from team mates, practical code related things as well as social.
